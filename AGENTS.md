# Harbors Notifications Kit repository instructions

Use the repository-local `kit-workflow` Skill for every product change. Base and PR target must remain `kit/notifications`; never target `main`.

Commit titles must use exactly one of `[Init]`, `[Feature]`, `[Bug]`, `[Docs]`, `[Refactor]`, `[Optimize]`, `[Test]`, or `[Chore]`, followed by a concise Chinese summary without a trailing period. `[Init]` is initialization-only.
