# Harbors Kit Registry branch instructions

This repository history is the `kit-registry` product index. Stable entry changes must arrive through a reviewed pull request; preview automation may only update preview metadata. Never add Kit executables or source archives.

Commit titles must use exactly one of `[Init]`, `[Feature]`, `[Bug]`, `[Docs]`, `[Refactor]`, `[Optimize]`, `[Test]`, or `[Chore]`, followed by a concise Chinese summary without a trailing period. `[Init]` is initialization-only.
